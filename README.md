# Typing-Speed-Test
Speed Typer is a Python-based CLI (Command Line Interface) application designed to test and improve typing speed and accuracy.

Features
Select typing duration: 30, 60, or 120 seconds
Generates a random paragraph based on selected time
Real-time typing test with countdown timer
Calculates Words Per Minute (WPM)
Counts Correct Words
Counts Wrong Words
Counts Missed Words
Calculates Accuracy Percentage
Analyzes typing style (Beginner, Average, Fast, Professional)
Technologies Used
Python
Random module
Time module
String manipulation
Functions and loops
How It Works
Run the program in the terminal.
Choose a typing duration (30, 60, or 120 seconds).
A random paragraph will be displayed.
Type the paragraph before the timer ends.
Receive a detailed performance report.
Output Metrics
WPM (Words Per Minute): Measures typing speed
Correct Words: Words typed correctly
Wrong Words: Incorrectly typed words
Missed Words: Words not typed
Accuracy: Typing precision percentage
Typing Style: Performance classification
Purpose

This project helps users improve typing speed and accuracy while demonstrating core Python concepts such as loops, functions, timers, and string comparison.
